World Wide Web - Main concepts

What is HTML

HTML hypertext markup language is the main type of document on the web. It serves as a glue of all other documents that can be found on the web photos, videos, texts, links (hyperlinks)...

Main pieces on the web are: me, my computer/browser, the internet, HTTP(the main protocol on the web) and servers.

HTML is made up of:
text content
markup
referernces to other documents
links to other pages
